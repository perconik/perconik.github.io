Requirements

  • Java Platform SE 7 (or higher)
  • Eclipse Platform 4.3.0 (or higher)
  • Eclipse Java Development Tools and Eclipse EGit (for proper activity logging)
  • UACA at least Spongebob & Patrick version (depends on Windows operating system and .NET 4.0)
  • Internet connection (may be required by some services depending on configuration)

Installation

Quick Installation Guide:

Eclipse → Help → Install New Software... → Available Software Sites → Add

Name
PerConIK Eclipse Integration
Location
http://perconik.github.io/update

(see Eclipse Plug-in Update Site for more details about current stable plug-in version)

Full Installation Guide:

  1. Add Eclipse Plug-in Update Site (see Quick Installation Guide above)
  2. Go to Help → Install New Software...
  3. Select PerConIK Update Site - http://perconik.github.io/update in the Work with field
  4. Check PerConIK Eclipse Installation and proceed with Next
  5. Accept the terms of license agreement and Finish the installation
  6. Plug-ins are not digitally signed, go ahead and install anyway
  7. Restart Eclipse when prompted

Installed plug-ins can be easily updated via:

Eclipse → Help → Install New Software... or Eclipse → Help → Check for Updates

Update steps are in fact the same as installation steps above.
